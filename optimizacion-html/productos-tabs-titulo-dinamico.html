🎯 Resumen del cambio

🔤 Título dinámico según tab activo

El <h1> de la página ya no muestra siempre "Nuestras Recetas". Ahora reacciona al tab seleccionado: cuando el usuario filtra por una categoría, el título cambia para reflejarla. El tab Todos sigue mostrando "Nuestras Recetas".

🎨 Color unificado en tabs de filtro

Los tabs "Para Celebrar" (amarillo), "Para Premiar" (azul) y "Para Complementar" (verde) pasaron a usar el color primario de la web (bg-primary — #7AB8BF), igual que el tab "Todos". Esto da coherencia visual y respeta el sistema de diseño existente.
